# 4Fun-Wealth-Management-Web

This repository contains two main parts:
1. **Interactive Wealth Management Mind Map**: The original React application mapping various financial advisory branches.
2. **Bourgouin & Associates Website (Worksite2)**: A premium, black-and-white minimalist landing page and corporate structure website integrating the mind map and a direct booking consultation calendar.

---

## Directory Structure

* `/` (Root): The original React & TypeScript mind map application.
* `/Worksite2`: The new self-contained Bourgouin & Associates website featuring the landing page, "About Us" director profiles, "Products" page, and integrated mind map with a direct booking consultation form.

---

## Getting Started

### 1. Running the Original Mind Map (Root)

To run the original application:

```bash
# Install dependencies
npm install

# Run the local development server (Default port: 3000)
npm run dev
```

### 2. Running Bourgouin & Associates Website (Worksite2)

To run the new website:

```bash
# Navigate to the website folder
cd Worksite2

# Install dependencies
npm install

# Run the local development server (Default port: 3001)
npm run dev
```

Open your browser to [http://localhost:3001](http://localhost:3001) to explore the Bourgouin & Associates website, scroll to view the mind map, and try the custom B&W consultation booking calendar.
